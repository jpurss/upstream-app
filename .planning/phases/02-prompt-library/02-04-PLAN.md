    Three exported functions:

    **`createPrompt(formData: FormData)`:**
    - Extract from formData: `title`, `description`, `content`, `category`, `capability_type`, `industry_tags` (comma-separated string → split into array), `use_case_tags` (comma-separated string → split into array), `target_model`, `complexity`, `input_schema`, `output_schema`
    - Get user via `await createClient()` → `supabase.auth.getUser()`
    - Check role: `user?.app_metadata?.role`. If not admin, also check `user?.is_anonymous && user?.user_metadata?.demo_role`. If not admin, return `{ error: 'Unauthorized' }`.
    - Insert: `supabase.from('prompts').insert({ title, description, content, category, capability_type, industry_tags, use_case_tags, target_model, complexity, input_schema: input_schema || null, output_schema: output_schema || null, created_by: user.id, status: 'active' })`
    - On error: return `{ error: error.message }`
    - On success: `revalidatePath('/library')`, `redirect('/library')`

    **`updatePrompt(promptId: string, formData: FormData)`:**
    - Same field extraction as create
    - Same role check
    - Update: `supabase.from('prompts').update({ ...fields, updated_at: new Date().toISOString() }).eq('id', promptId)`
    - On error: return `{ error: error.message }`
    - On success: `revalidatePath('/library')`, `revalidatePath(\`/library/${promptId}\`)`, `redirect(\`/library/${promptId}\`)`

    **`deprecatePrompt(promptId: string)`:**
    - Role check same as above
    - Update: `supabase.from('prompts').update({ status: 'deprecated' }).eq('id', promptId)`
    - On error: return `{ error: error.message }`
    - On success: `revalidatePath('/library')`, `revalidatePath(\`/library/${promptId}\`)`, return `{ success: true }`
