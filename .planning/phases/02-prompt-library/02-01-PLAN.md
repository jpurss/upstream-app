    **Create `lib/data/prompts.ts`:**
    Server-side data-access functions using the Supabase server client (`createClient` from `@/lib/supabase/server`):

    - `fetchAllActivePrompts()`: Queries `prompts` table with `.eq('status', 'active')`, ordered by `avg_effectiveness` descending. Returns `Prompt[]`.
    - `fetchPromptById(id: string)`: Queries single prompt by UUID. Returns `Prompt | null`.
    - `searchPrompts(query: string)`: Uses `.or()` with `ilike` on title, description, content fields (per RESEARCH.md Pitfall 1 — ilike fallback for v1 with 18 prompts). Returns `Prompt[]`.
    - All functions import `Prompt` from `@/lib/types/prompt`.
    - All functions use `await createClient()` from `@/lib/supabase/server`.

    **Update `app/layout.tsx`:**
    - Add `import { Toaster } from 'sonner'` at top.
    - Add `<Toaster />` inside the body element, after the ThemeProvider children but before closing body tag. This makes toasts available globally.

    - Wrap the `{children}` inside the `<main>` element with `<NuqsAdapter>{children}</NuqsAdapter>`. This enables nuqs `useQueryState` hooks in all app routes.
